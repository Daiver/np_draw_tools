# np_draw_tools
Different numpy/opencv based draw tools


# Installation

## Pip

```
pip3 install git+https://github.com/Daiver/np_draw_tools
```

## Manual
```
git clone https://github.com/Daiver/np_draw_tools
cd pygeom_tools
python3 setup.py install
```
